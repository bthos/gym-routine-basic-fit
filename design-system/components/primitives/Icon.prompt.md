Thin-line stroke icon — a Feather/Lucide (ISC) subset matching basic-fit.com's icon style. Never use emoji as icons; use these. Color comes from the parent's `color` (currentColor).

```jsx
<Icon name="search" size={20} />
<span style={{ color: "var(--bf-purple)" }}><Icon name="map-pin" /></span>
```
